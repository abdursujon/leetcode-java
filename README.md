# LeetCode Java

A practice repo for interview preparation and developing problem-solving skill in Java. 
Each file isolates **one problem-solving pattern** so we can learn the patterns and techniques in a small scope.

Suggested order of learning since each topic builds on the ones above it. The patterns that will be included are listed below. 
>Note: You can click any topic below to view the code directly (if the patterns are done).
1.  Hashing (HashMap, HashSet) x
2.  [Two Pointers](problem-patterns/two_pointers/TwoPointers.java) (array, string - usually sorted) x
3.  SlidingWindow (array, string + HashMap/HashSet) x
4.  PrefixSum (array + HashMap)
5.  BinarySearch (sorted array)
6.  FastAndSlowPointers (linked list, array)
7.  LinkedListReversal (linked list)
8.  MergeIntervals (array of intervals - sort first)
9.  CyclicSort (array with values 1..n)
10. MonotonicStack (stack / ArrayDeque)
11. TopKElement (heap / PriorityQueue)
12. TreeBFS (tree + queue / ArrayDeque)
13. TreeDFS (tree + recursion/stack)
14. GraphTraversal (adjacency list Map/List + queue/stack + visited Set)
15. GridTraversal (2D array + queue/recursion + visited)
16. TopologicalSort (adjacency list + queue + in-degree array)
17. BackTracking (recursion + list for current path)
18. DynamicProgramming1D (1D dp array)
19. DynamicProgramming2D (2D dp array)
20. Greedy (array - often sorted, sometimes heap)

## Technique to solve an unknow problem 
> R2PSDIT 
- R2: Read the question twice 
- P: Recognise the pattern 
- S: Write sudo code 
- D: Think about the data structure needed 
- I: Implementation 
- T: Test the implementation
