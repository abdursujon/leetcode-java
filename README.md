# LeetCode Java

A practice repo for interview preparation and developing problem-solving skill in Java. 
Each file isolates **one problem-solving pattern** so we can learn the patterns and techniques in a small scope.

Suggested order of learning since each topic builds on the ones above it.

## Technique to solve an unknow problem 
> R2PSDIT 
- R2: Read the question twice 
- P: Recognise the pattern 
- S: Write sudo code 
- D: Think about the data structure needed 
- I: Implementation 
- T: Test the implementation
